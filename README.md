# to-DOs

This is a task management web application built with React, part of the React.js course from Platzi. You can use it to create, edit, and organize your tasks.

**Live Demo:**  
[Click here to try the app](https://juanjomayotte.github.io/to-DOs/)

## Features

- **Task Management:** Create, edit, and delete tasks.
- **Search:** Filter tasks in real-time by name using the search bar.
- **User-Friendly Interface:** Clean and simple design.

## Technologies Used

- **Frontend:** React, HTML, CSS, JavaScript
- **React Libraries:** react-icons, react-scripts

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/juanjomayotte/to-DOs.git
   ```
2. Navigate into the project directory:
   ```bash
   cd to-DOs
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm start
   ```
